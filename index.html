UX design

UX design signifie User Experience Design. En Français, il s’agit du design de l’expérience utilisateur.

UI design

L’UI design se rapporte à l’environnement graphique dans lequel évolue l’utilisateur d’un logiciel.

SEO

L'optimisation pour les moteurs de recherche, référencement naturel ou SEO (pour Search Engine Optimization), est un ensemble de techniques visant à optimiser la visibilité d'une page web dans les résultats de recherche.
